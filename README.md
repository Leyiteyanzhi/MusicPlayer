## 功能说明

### 歌曲切换
  
  - [x] 点击上/下首切换按钮，歌曲将进行切换
  - [x] 切换效果为唱片匀速进入页面中心，被切换歌曲唱片匀速退出
  - [x] 完成背景图片，唱片图片，歌词切换
  
### 暂停/播放

  - [x] 点击暂停按钮，歌曲在当前进度展厅
  - [x] 唱片停止旋转，按钮图标相应转换为播放按钮
  - [x] 点击播放按钮，歌曲继续播放
    
### 进度条

  - [x] 点击进度条任意位置，歌曲将跳转当前进度
  
### 循环/单曲

  - [x] 点击循环按钮，歌曲按照播放列表循环播放
  - [x] 点击单曲按钮，歌曲将不断单曲循环
    
### 歌词显示

  - [x] 点击唱片，进入歌词显示
  - [x] 歌词根据进度不断呈现，并在当前演唱句醒目标出
  - [x] 可拖动鼠标，选择相应歌词，即歌曲转到当前歌词时间继续播放
    
### 歌单列表

  - [x] 展示所有歌曲
  - [x] 点击可选择播放歌曲
  - [x] 当前播放歌曲将在列表中以小广播形式展现
